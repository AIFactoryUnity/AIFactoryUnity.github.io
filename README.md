# AI Factory Unity Website

This repository contains the public website for **AI Factory Unity (AIFU)**.

The repository is currently public because it is deployed through GitHub Pages. Public access to this repository does not mean that the content is open-source or free to reuse.

## Website

AI Factory Unity’s public website is available here:

https://aifactoryunity.github.io

## About AI Factory Unity

AI Factory Unity is an early-stage AI startup building AI-assisted software for industrial manufacturing environments.

The website is used for early public communication, startup presentation, team visibility, funding acknowledgement, and basic online presence.

## Repository Contents

* `index.html` — main website page
* `assets/brand/` — AI Factory Unity brand assets
* `assets/funding/` — SIBB Incubator, ESF+/European Union, and Berlin Senate logos used for required public funding acknowledgement
* `assets/team/` — team photos used only for the AI Factory Unity team section
* `README.md` — repository information
